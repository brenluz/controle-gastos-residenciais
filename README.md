# Controle de Gastos Residenciais

Sistema para controle de gastos residenciais com cadastro de pessoas,
cadastro de transações (receitas/despesas) e consulta de totais por pessoa.

Projeto de teste técnico — back-end em **.NET 8 / C#**, front-end em
**React + TypeScript**, com persistência dos dados em **SQLite**.

> Documentação de execução será detalhada ao final da implementação.
